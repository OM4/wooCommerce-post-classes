Adds additional information (such as shipping class) to the WooCommerce product display.

== Description ==

This plugin adds the additional following information to the DOM classes for each product:

* Shipping Class Name and ID (shipping-class-* and shipping-class-id-*)

If you have any other suggestions for other information that could be included, please [add your suggestion via GitHub](https://github.com/OM4/woocommerce-post-classes/issues).

== Installation ==

1. Download the latest stable version of the plugin from https://github.com/OM4/woocommerce-post-classes/archive/release.zip.
1. Install this plugin via WordPress Dashboard, Plugins, Add New, Upload.
